# Raven’s Cozy Keepsakes

A storybook memory card game featuring Raven, Willow, Salem, and Bo. Match characters with their favorite cozy items across multiple levels. Built with HTML, CSS, and JavaScript, and designed for GitHub Pages.

## Play locally
Because this uses ES modules, run a local server (not by double-clicking the file).

### Option A (Python)
```bash
python -m http.server 8000
```
Then open: http://localhost:8000

### Option B (VS Code)
Use the “Live Server” extension.

## Deploy to GitHub Pages
1. Push this folder to a GitHub repo.
2. Repo → Settings → Pages
3. Source: **Deploy from a branch**
4. Branch: **main** / **root**
5. Save

## Characters & story matches
- Raven → Security Blanket
- Willow → Cat Bed
- Salem → Rock
- Bo → Feather
